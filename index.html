What is static website?

A Static Website contains fixed content and It won't change by any means unless it is updated by the owner of the website. The content in the Static website is shown as-it-is for all users even they are in different locations.

What is dynamic website?

A Dynamic Website requires server-side scripting and was written in some languages like PHP,Python,ASP.Net. It content of the Dynamic Website changes according to the user inputs. Examples facebook,Twitter etc.
